TypeScript

O TypeScript (TS) é uma linguagem desenvolvida pela Microsoft que funciona como uma camada de segurança para o JavaScript (JS).

JavaScript: É dinâmico. Você pode dizer que uma variável é um número e depois mudar para um texto. Isso causa erros inesperados.

TypeScript: É estático. Você define que um dado é um number, e se tentar passar um string, o código nem compila.

Resultado: Menos bugs em produção e um código muito mais fácil de ler e manter.

1- instalar o compilador 

 npm install -g typescript

2- Versão 

tsc -v
